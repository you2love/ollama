🧠 模型管理

模型下载、列表、自定义与部署

📥 模型下载

下载官方模型

ollama pull llama3.2        # 最新版
ollama pull llama3.2:1b   # 指定参数
ollama pull qwen2.5        # 通义千问
ollama pull phi3          # 微软小模型
ollama pull mistral       # 欧洲开源

查看可用模型

ollama list                 # 已下载
ollama search llama       # 搜索可下载

📊 主流模型对比

模型 参数 场景 推荐配置
Llama 3.2 1B/3B 通用对话 ≥8GB VRAM
Qwen 2.5 0.5B-72B 中文优化 ≥16GB VRAM
Phi-3 3.8B 轻量部署 ≥4GB VRAM
Mistral 7B 代码任务 ≥8GB VRAM

⚙️ 自定义模型 (Modelfile)

创建自定义提示词

# 创建 Modelfile
FROM llama3.2

PARAMETER temperature 0.7
PARAMETER top_p 0.9

SYSTEM """
你是一个专业的技术文档助手。
用简洁清晰的语言回答技术问题。
"""

加载自定义模型

ollama create my-assistant -f ./Modelfile
ollama run my-assistant

🗂️ 模型管理命令

查看与运行

ollama list        # 已下载列表
ollama ps          # 运行中的模型
ollama run llama3.2

删除与复制

ollama rm llama3.2         # 删除
ollama cp llama3.2 my-copy  # 复制
